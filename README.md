# Germany Relocation Timeline

Personal checklist app tracking a 24-month immigration, marriage, and employer compliance plan for a US citizen relocating to Germany.

**Built with:** HTML · CSS · Vanilla JS · Cloudflare Pages · Cloudflare Workers KV

**Features:**
- Collapsible phases with progress tracking
- Per-step notes (click to expand, auto-saves)
- Shared state between two users via KV storage
- Works on any device — no app install needed
